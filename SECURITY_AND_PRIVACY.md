# SECURITY_AND_PRIVACY

## 公开仓库不应包含的内容

- 真实 API Key、`.env`、账号密码或访问令牌。
- 学生真实姓名、学校、班级、学号、联系方式、身份证号等可识别信息。
- 原始学生图片、正脸照片、完整学生作文原文。
- `data/raw/`、完整训练集、完整外部验证集、私有 OCR 全文。
- 内网演示地址、本地服务器账号、个人微信路径或聊天文件路径。

## 系统使用边界

AIGC 风险指数表示模型参考概率，不代表 AI 内容占比，不作为学生违纪判定或处分依据。教师应结合草稿、访谈、学生 AI 使用声明和过程材料综合判断。

## 图片与 OCR

图片作业识别前请遮挡可识别信息。OCR 结果可能存在漏字、错字或段落顺序错误，分析前请教师确认和必要修正。系统默认不保存原始图片，看板只保存文本摘要、hash 和分析结果。
