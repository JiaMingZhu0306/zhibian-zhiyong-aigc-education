# 隐私合规与使用边界说明

## 一、基本原则

本系统服务于中学生 AIGC 作业规范使用教育和 AI 素养培养。系统输出的是教学辅助信息，不是纪律、司法或人格评价结论。所有风险提示仅供教师参考，不作为纪律处分依据。

## 二、禁止上传的信息

- 学生真实姓名。
- 学校、班级、学号等可直接识别学生的信息。
- 身份证号、手机号、家庭住址等敏感信息。
- 学生正脸照片。
- 未经授权的完整学生隐私原文。

## 三、图片上传与 OCR 边界

图片作业识别和批量作业导入前，请教师先遮挡学生姓名、班级、学号等可识别信息。OCR 识别可能存在漏字、错字或段落顺序错误，分析前请教师确认和必要修正识别文本。

系统不默认保存原图。批量 OCR 结果仅供教师本地核对，不公开展示学生原文。看板只保存文本摘要、hash、风险等级、作业类型、年级、来源说明等分析结果，不保存完整学生原文。

如教师本地生成 `data/local/private_ocr_texts.csv` 用于核对 OCR 全文，该文件属于本地私有文件，不进入最终提交包。

## 四、AI 生成内容标注

系统生成的声明模板、反思问题、班会方案和教师建议均应标注“AI 辅助生成，仅供教师参考”。教师在正式使用前需要审核、修改和本地化。

## 五、教师综合判断

AIGC 风险指数表示模型参考概率，不代表文本中 AI 生成内容占比。教师应结合草稿、访谈、学生 AI 使用声明、修改记录和课堂过程性观察综合判断。系统不替代教师教育判断权。

## 六、最终提交包隐私要求

最终提交包不得包含：

- `data/raw/`
- 原始学生图片
- `data/local/private_ocr_texts.csv`
- `data/local/uploaded_images/`
- `data/local/ocr_cache/`
- 完整学生作文外部验证 CSV
- 完整 public_train/public_val/public_test
- 真实 API Key

本内容由 AI 辅助生成，仅供教师参考。
